A bot for Discord, written in TypeScript. Uses the discord.js library and a connection to a MongoDB database.

Weather App

A weather app written in JavaScript. Allows the user to fetch their weather based on location, or by inputting coordinates.

Magic: The Gathering - Card Search

A Magic: The Gathering based web app that was written with TypeScript. Allows the user to search for a card by name and optionally by card set. Fetches data from an API and renders it in a readable format.

A Magic: The Gathering themed "Rock Paper Scissors" clone, written in TypeScript. Determines the winner of each round and tracks the scores.

Card Counting Trainer

A web app designed to help the user learn how to count cards in BlackJack. Includes three levels of increasing difficulty. Written with JavaScript.

Smash A Bug

A programming themed "whack a mole" clone written in JavaScript. Allows the user to set a goal for the number of bugs smashed, and tracks progress toward that goal.
